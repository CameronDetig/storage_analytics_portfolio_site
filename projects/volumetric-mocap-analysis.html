Volumetric Motion Capture Analysis Tool

Blender Addon for Biomechanical Skeleton Extraction

Project Overview

This Blender addon enables the extraction of articulated skeletons from volumetric motion capture data. Developed as part of an Immersive Media Research Experience for Undergraduates (REU) at Georgia State University, the tool is intended for biomedical analysis to identify strenuous movements from recordings of athletes, as well as for use in film and animation contexts.

The project was mentored by Dr. Zing Yu.

Technical Implementation

Volumetric Motion Capture

Traditional motion capture uses markers placed on a subject's body to track movement. Volumetric motion capture, in contrast, uses multiple cameras to capture the full 3D volume of a subject without markers, producing detailed mesh sequences. This addon bridges the gap between volumetric capture data and traditional skeleton-based animation workflows.

System Overview

Skeleton Extraction Process

The tool uses the Iterative Closest Point (ICP) algorithm to align joint markers to the volumetric mesh data frame by frame. This allows researchers and animators to extract meaningful skeletal motion data from markerless capture sessions.

  • ICP Algorithm - Utilizes Christopher Gearhart's ICP implementation for precise point alignment
  • Frame-by-Frame Processing - Processes mesh sequences to track joint positions over time
  • Blender Integration - Seamlessly integrates into Blender's 3D workflow

Applications

  • Biomedical Analysis - Identify strenuous movements and analyze athlete performance
  • Film & Animation - Extract skeleton data for use in traditional animation pipelines
  • Research - Enable biomechanical studies using volumetric capture data
